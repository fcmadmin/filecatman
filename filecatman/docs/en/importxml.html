Import XML

With this wizard, a Filecatman XML file (*.xml) can be selected and the data imported. The types of data stored in the file, can be items, categories, relations, item types and taxonomies. At the end of the import process a finish page will be shown.

Results

When the data has been imported, a finish page like the one below will be displayed, showing the time elapsed, and the counts of how much of the data was imported.
