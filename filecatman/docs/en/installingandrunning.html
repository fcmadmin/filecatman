Installing and Running

Install in a venv (Linux terminal):

python -m venv ./venv
source venv/bin/activate
pip install .
filecatman

Install in a venv (Windows admin CMD):

python -m venv .\venv
.\venv\Scripts\activate
pip install .
filecatman

Install in a venv (Mac terminal):

python -m venv ./venv
source venv/bin/activate
pip install .
filecatman

Linux install using new method (requires pip):

sudo pip install .
filecatman

Linux install using deprecated method (requires setuptools):

python3 setup.py clean -a
python3 setup.py build
sudo python3 setup.py install
sudo python3 setup.py install_data
python3 setup.py desktop
filecatman

Windows in admin CMD (requires python 3.10+):

pip install .
(add the installation directory to the Windows PATH)
filecatman

Mac in terminal (requires python 3.10+):

sudo python -m pip install .
filecatman

Or run from source:

pip install -r requirements.txt
python maindev.py
