Open MySQL Database

Connect to a MySQL server, and open an existing database.

Hostname

The host where the MySQL server is running.

Port

The port number to use for the connection, the default port number is 3306.

Username

The user name of the MySQL account you want to use.

Password

The password of the MySQL account.

Database

The name of the existing database.

Open database on startup.

Clicking this checkbox will enable the option to automatically open the database at startup, skipping the startup wizard. This setting can be changed later in the Preferences Dialog.
