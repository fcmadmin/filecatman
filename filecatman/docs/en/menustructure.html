The File Menu

The “File” menu show actions for selecting a database file or connection, as well as importing data from a file and exporting data to a file, inserting data into the database, or exiting the application.

File->New Item (Ctrl+N)

Opens the “New Item Dialog”, allowing the user to add a new item to the database.

File->New Category (Ctrl+Shift+N)

Opens the “New Category Dialog”, allowing the user to add a new category to the database.

File->Import XML

Opens the “Import XML Wizard”. Data can be imported from a Filecatman XML file (*.xml).

File->Export XML

Opens the “Export XML Wizard”. Data can be exported to a Filecatman XML file (*.xml).

File->New Database

Closes the current database, and brings up the “Startup Wizard”, where the user can create a new database, either a SQLite database file (*.db) or a MySQL database connection.

File->Open Database

Closes the current database, and brings up the “Startup Wizard”, where the user can open an existing database, either a SQLite database file (*.db) or a MySQL database connection.

File->Close Database

Closes the current database, and brings up the “Startup Wizard”.

File->Exit Program (Ctrl+Q)

Quits the application.
