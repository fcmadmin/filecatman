Why categorization?
How can Filecatman make categorization useful?

Categorization is the core purpose of Filecatman. Filecatman allows data to be represented by multiple categories so that it can be better sorted and searched for and can be visualized in multiple folders. Filecatman provides the means for classifying files into however many categories the user specifies. Filecatman has the tools for making categorization an effective strategy.

Why categorization?

Categorized files are easier to find and easier to sort.
