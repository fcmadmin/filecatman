Removing From System

If installed in pip:

pip uninstall filecatman

else you will have to manually delete the Filecatman related files from your system. The main files are normally stored in one of the following places (Linux):

/usr/local/lib/python3.x/dist-packages
/usr/lib/python3/dist-packages
/usr/local/lib/python3.x/site-packages
/usr/lib/python3/site-packages

Example:

sudo rm -r /usr/local/lib/python3.3/dist-packages/filecatman*

To find and remove the remaining files use the locate command and to ensure up-to date information also run updatedb:

sudo updatedb
locate filecatman
