Database Options

After creating a database, this page will be displayed. Here you will choose the folder that will hold the files for your project.

Open database on startup.

Clicking this checkbox will enable the option to automatically open the database at startup, skipping the startup wizard. This setting can be changed later in the Preferences Dialog.

Data Directory

This is the folder that will hold the data files for your project. The files correspond to the items that you will insert into the database. If using SQLite, it's probably best to keep your database file close to your data directory. This setting can be changed later in the Preferences Dialog.

Category Descendant Levels

The maximum number of levels from a root category that a child category is allowed to descend.

Note

Increasing the level will make it take longer to load the category hierarchy.
