Filecatman Online

A file categorization management program, designed to make sorting, categorizing and accessing data simple and easy.

When creating a project, Filecatman saves it in a database, either as an SQLite file, or as a MySQL server connection.

Filecatman is written in the Python programming language using the Qt application UI framework and the PySide Python bindings. It is released under the GPLv3 licence.
