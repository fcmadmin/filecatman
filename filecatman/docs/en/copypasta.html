Copypasta

This feature allows for the quick copying of item details to clipboard. When an item is copied, the name, source and description are combined into a single text that can be conveniently pasted into a text field in a different program. It can be used on a single item or multiple items from the main table. If used on a category. All items in that category will be copied. Optionally the text can be automatically formatted in a specific way, see the Preferences Dialog for more details. The action can be found under the Table Menu. The default shortcut for Copypasta is Ctrl+Alt+C.

Usage Example

When selecting a row or multiple rows in the main table. The option for Copypasta becomes available. When triggered, the data about the selection is copied to the clipboard. The data can then be pasted anywhere.
