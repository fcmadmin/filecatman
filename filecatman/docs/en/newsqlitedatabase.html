New SQLite Database

Here the user can select a directory where the new SQLite database file will be saved. After creating a database, the Database Options page will be shown.

Location for the new SQLite database

Pressing the “Browse...” button brings up a file dialog, where the user can look through their directory structure and select which one will contain the new database file.
