Chapter 5. Filecatman Main Window

After a database has been loaded from the Startup Wizard, this window will open. It forms the main access point for most of the program's actions, and it provides the table views for displaying the project's data.

1. Menu Bar

The region of the application window where most actions are available in drop down menus.

2. Tool Bar

Commonly used actions are available here for quick access, as well as a search box to find items in the database that match keywords.

3. Items Menu, Categories Menu

Here you can select either items or categories to be displayed in the main table, as well as the type of items/categories. The last search results, or the list of items with missing files can be selected here.

4. Main Table

This is the main view of the program, where items and categories are shown. Rows can be selected with a single mouse click, or a multi highlighted selection can be made by holding the Shift or Ctrl key. With the selection, more actions become available, allowing you to access, edit, delete, copy etc... the selection. Rows can also be checked by clicking the checkbox or using the checkbox keyboard shortucts. Only checked rows have the bulk actions available to be used on them.

5. Details Tab

Displays all details about the selected row in the table. Any available source or folder location links can be opened.

6. Relations Tab

Displays a table of the selected row's relations. Options for deleting and opening the selected relation are available, as well as deleting all relations for the selected row. Opening a relation will either display items with the selected category or launch the item in the default application, depending on whether item or category relations are shown.
