How can Filecatman make categorization useful?

Filecatman provides a lot of tools for categorizing data, searching for data, and using data. Here are some if it's features:

Copypasta

This feature allows for the quick copying of item details to clipboard. When an item is copied, the name, source and description are combined into a single text that can be conveniently pasted into a text field in a different program. See Copypasta for more details.

View

View the selected row. If the table is displaying items, the file will be opened in the default application, else if categories are shown, viewing the category will display all items in that category.

Copy File Path

Copy the file path of the selected item. With the file path, the user can paste it in a select file dialog in a different application, this allows for quick access to files. If the item is not a file, but a weblink, then the URL source will be copied.

Create Symbolic Links

Create a symbolic links hierarchy from the item-category relations. Creating links allows for the item-category relations to be viewed in the system's file manager. See Create Symbolic Links for more details.

Advanced Search

An advanced searching dialog for customizing the search query with lots of parameters. With the item-category relations, a lot of search options become available, which makes looking for related files easier than ever. Filecatman can sort files based on many variables. It can search for files with a specific list of categories, as well as sort files based upon their date of publication or what type of file they are. See Advanced Search for more details.

Import, Export

Data can be exported to an XML file. The XML could then be imported back into a project, or it could be packaged with the data folder and the quick installers. The quick installer does the same function as the Create Symbolic Links Wizard, but doesn't require Filecatman. The quick installers are included in the source code.
