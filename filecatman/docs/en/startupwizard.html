Database Options

This is the first window that will appear when the user opens the program for the first time. It displays a menu with options for creating a new database project, or opening an existing database project. The supported databases that the project can be, are either SQLite or MySQL.

Welcome Page

The page that is shown first with the startup wizard. It has two menu buttons, one for creating a new database and one for opening an existing database.

New Project

Clicking this brings up a sub menu with options for creating an SQLite database file or a MySQL database connection. When the choice is made, the wizard moves on to the next page.

Open Project

Clicking this brings up a sub menu with options for opening an existing SQLite database file or a MySQL database connection. When the choice is made, the wizard moves on to the next page.

Help

Brings up the help browser, where this documentation can be read.

Exit

Quits the application.
