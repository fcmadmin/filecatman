New Category

This dialog presents the dialog for inserting a new category into the database. Information about the category, such as the name, taxonomy and description can be specified. When the user accepts the dialog, the category is inserted into the database.

Main Details

The form where important details about the category are specified.

Name

Type a name for the category.

Slug

This is used as an identifier for the category, it's used mainly by websites for URL naming. If left blank, the name will be sluggified when the dialog is accepted.

Taxonomy

This is a classification for a range of categories. The supported Taxonomies can be modified in the Preferences Dialog.

Parent

This specifies that the category will be a child category of another category in the same taxonomy. Taxonomies can have parents disabled if the option is set in the Preferences Dialog.

Description

This is where a description of the category can be written.
