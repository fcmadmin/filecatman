The View Menu

Customize which elements of the GUI are to appear, and toggle full screen mode.

View->Toolbars

Toggle which toolbars are to be displayed.

View->Selection Details

Toggle the tabs at the bottom of the screen which display information about a selected table row.

View->Sidebar

Toggle the item types/taxonomies menu at the side of the screen.

View->Statusbar

Toggle the statusbar which appears at the bottom of the screen and gives tips about an action being hovered over.

View->Full Screen (F11)

Toggle full screen mode for the main window.
