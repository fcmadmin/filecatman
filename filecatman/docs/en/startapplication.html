Usage in the command line
options

The quickest way to open Filecatman is to launch the shortcut in the application menu. Filecatman can also be started by a terminal command. Some command line options can be specified.

Usage in the command line

filecatman [options]

options

Optional arguments. You can specify some switches to setup the application. Option list:

  • --help or -h display a list of command line options and exit.

  • --version or -v show program's version number and exit.

  • --database or -db specify a filepath to load an SQLite database. E.g. filecatman --database test.db

  • --loglevel or -L set the log level: none, info, warning, error, critical or debug. E.g. filecatman --loglevel debug

  • --quiet or -q sets the log level to 'none', this is the same as -L none.
