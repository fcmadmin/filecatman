Create Symbolic Links

This wizard can create a symbolic links hierarchy from the item-category relations. The symbolic links are saved in the directory that the user specifies. Creating links allows for the item-category relations to be viewed in the system's file manager. Details about the creation of the links are shown in a status page at the end of the process, more details can be found in log files created in the symbolic links directory.

Symbolic Links Directory

The location where the symbolic links will be created.

Links to be created

The estimated count of the amount of links that will be created.

Overwrite existing links

When checked, links that already exist will be deleted and then recreated. If not checked, existing links won't be overwritten.
