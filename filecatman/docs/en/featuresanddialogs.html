New Item

This dialog presents the dialog for inserting a new category into the database. To select a file, you can use the upload dialog, which will copy the file into the project's data directory, or use the select file dialog to choose an already existing file from the data directory. Information about the file, such as the source, data of publication and description can be specified. The relations tab, is where the item can be categorized. When the user accepts the dialog, the item and it's relations are inserted into the database. An error message is displayed if the item is a duplicate.

Caution

When uploading Webpages that have a data folder, make sure the data folder shares the same name as the file, E.g. “thispage.html” and “thispage_files”.

Main Details

The main form, where important details about the item are specified.

Type

Choose the Item Type of the file which is to be selected, or choose a Weblink to insert a URL. The supported Item Types can be modified in the Preferences Dialog.

File, Name

The file that this item will be associated with. Or if it's of type Weblink, the name of the Weblink.

Caution

The name must not contain any of these illegal characters: (' " ` $). Curly quotes (‘’ “”) can be used instead of straight quotes.

Upload

Opens the upload dialog, so the user can select a file to copy into the data directory.

Browse...

Opens the select file dialog, so the user can select an existing file from the data directory.

Source

The URL of the file or weblink. This field is mandatory for items of type Weblink.

Date

The date of publication of the file or weblink. An option for a null date is available. The default date is that when the “New Item Dialog” was opened.

Description

This is where a description of the item can be written. It could also be an excerpt.

Relations

In this tab the category relationships for the new item can be selected.

Taxonomies Menu

Here you can select which type of categories are to be displayed in the categories tree to the right. If the taxonomy is in “Tags Mode”, the categories will be displayed in a list. The supported Taxonomies can be modified in the Preferences Dialog.

Categories Tree

A tree of the available categories for the selected taxonomy will be shown here. Categories can be selected by clicking their checkbox. To the bottom of the tree is a form to insert a new category into the database. If the taxonomy is in “Tags Mode”, a blank list will be presented, and the form below the list will append or remove tags from the list. Tags are only inserted into the database after the “New Item Dialog” has been accepted.
