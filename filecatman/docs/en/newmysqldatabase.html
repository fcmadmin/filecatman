New MySQL Database

Connect to a MySQL server, and create a new database. After creating a database, the Database Options page will be shown.

Hostname

The host where the MySQL server is running.

Port

The port number to use for the connection, the default port number is 3306.

Username

The user name of the MySQL account you want to use.

Password

The password of the MySQL account.

Database

The name to be given to the new database.
