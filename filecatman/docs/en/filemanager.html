File Manager

This dialog allows the user to browse through their project's data directory. It has the ability to warn the user when a file doesn't have an item, and there is an advanced rename action, that can rename a webpage's files folder and change the links to the folder within the webpage.

Upload File

Opens the upload dialog, so the user can select a file to copy into the data directory.

Open

Open the selected file. The file will be launched in the default application.

Rename

Rename the selected file. If the file is a webpage, the data folder linked to the webpage will be renamed, this only works when the webpage's data folder shares the same name as the webpage, E.g. “thispage.html” and “thispage_files”.

Delete

Delete the selected file, will also delete a webpage's data folder. A confirmation message will be shown, including the option to delete the corresponding item.

Warning

The deletion of items and files cannot be undone.
