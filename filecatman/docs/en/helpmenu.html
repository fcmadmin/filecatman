The Help Menu

Here the user can open the “Help Browser Window”, and view information about this application.

Help->Help Contents (F1)

Opens the “Help Browser Window”, which displays a text browser for the reading of this documentation.

Help->About

Opens the “About Dialog”. Here information about the application such as the title and version are displayed, as well as a summary of the license.

Help->About Qt

Opens the “About Qt Dialog”. Information and licensing about the Qt cross-platform application UI framework is displayed.
