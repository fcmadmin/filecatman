Advanced Search

An advanced searching dialog for customizing the search query with more parameters. With the item-category relations, a lot of search options become available, which makes looking for related files easier than ever. Filecatman can sort files based on many variables. It can search for files with a specific list of categories, as well as sort files based upon their date of publication or what type of file they are. The results appear in the Main Window after confirming the search.

Keywords, Phrase

Search for items that have or don't have keywords or a phrase. The search can look in any field, or in a single field.

Date

Select items matching a date criteria. The date can be an exact date, a date range, or a null date.

File Type

Pick out or omit items that have a certain file type.

Taxonomy

Select items that have or don't have relations with this taxonomy. Selecting a taxonomy enables the parameter below.

Category

With a taxonomy criteria, also include a category.

Relations

Add multiple category arguments to the search query.
