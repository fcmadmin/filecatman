The Table Menu

Actions relating to the main table and a current table selection are here. This menu can also be brought up by right clicking on the main table.

Table->Copy File Path (Ctrl+C)

Copies the selection's file path if it's a file, or source if it's a weblink. This tool is useful if you need to quickly access the file in a different application, such as a select file dialog. Also works with a multi highlighted selection.

Table->Copypasta Selected (Ctrl+Alt+C)

Copies details about the selection to clipboard. The details that can be copied are the name, source and description. Also works with a multi highlighted selection. See Copypasta for more details.

Table->View Selected (Ctrl+Return)

View the selected row. If the table is populated with items, then viewing the row will launch the selected item in the default application, or if the table is populated with categories, viewing the category will display all items with the selected category.

Table->Edit Selected (F2)

Edit an item, a category or a selection of items. If multiple items are selected, the “Bulk Edit Items Dialog” will open, allowing the user to apply changes to every item in that selection, such as inserting or deleting relations.

Table->Delete Selected (Ctrl+Del)

Delete the selection. Delete the selected row or rows. A confirmation dialog will be displayed including the option to delete the associated files with the items.

Warning

The deletion of items and files cannot be undone.

Table->Find (Ctrl+F)

Shows the “Find Bar” at the bottom of the main table. Here you can search in the currently displayed rows for the matching phrase. Options to scroll through the results and highlight all search results are available.

Table->Select All (Ctrl+Alt Gr+A)

Highlight all rows in the main table. This process can be slow on large amounts of data. If you want to apply a checkbox action to all rows, use the checkbox actions menu instead.

Table->Checkbox Actions

Includes options to check rows in the table. Actions for checking rows, unchecking rows, and invert checking rows are available to be used on a highlighted selection or the entire table.

Table->Open Folder (Ctrl+Home)

Opens the data folder belonging to the current project. The data folder contains the files that are associated with the items in the database.

Table->Refresh Table (F5)

Refreshes the data in the table. Any checked selection will be reset.

Table->Bulk Actions

Lists actions that can be applied to the checked selection in the main table.
