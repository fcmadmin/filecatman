The Tools Menu

Actions that have a unique function are in this menu.

Tools->Database Info

Opens the “Database Info Dialog”, which displays properties about the current database, such as the counts of items, categories and relations, the tables that exist, and the database driver version.

Tools->File Manager

Opens the “File Manager Dialog”. This is an extra feature that lets the user quickly browse through their project's data directory. It can tell the user when a file doesn't have an associated item, and has an advanced rename action, that can rename a webpage's files folder and change the links to the folder within the webpage.

Tools->Advanced Search (Ctrl+S)

Opens the “Advanced Search Dialog”. This dialog provides the user with more parameters to customize in the search query.

Tools->Create Symbolic Links

Opens the “Create Links Wizard”. Here the item-category relations can be created and represented as a symbolic links hierarchy within a directory on the computer.

Tools->Item Checker

Scans each item in the database to see if it has an associated file. Any items with missing files can be displayed from the sidebar menu.

Tools->Link Checker

Every item that has a source will be checked to see if the source returns an error code when opened. Any items with broken links can be displayed from the sidebar menu.

Tools->Recount Relations

Recount all items in each category to check the counts are valid.

Tools->Show Log File

Open the log file in the default application.

Tools->Open Config Folder

Open the configuration folder in the system's file manager.

Tools->Vacuum Database

Vacuum the current database to reclaim free space and reduce file size. More information.

Note

Only available on an SQLite database.

Tools->Delete All Data

Deletes all items, categories and relations from the current database. A confirmation dialog will be displayed including the option to delete all files in the data folder.

Warning

The deletion of data or files cannot be undone.

Tools->Drop Database

Deletes the current database and returns to the “Startup Wizard”. If the database is a MySQL connection, then the database is dropped. Else if the database is a SQLite file, the file is deleted. A confirmation dialog will be displayed including the option to delete all files in the data folder.

Warning

The deletion of the database or files cannot be undone.
