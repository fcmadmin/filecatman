The Edit Menu

Actions for opening the “Preferences Dialog” where the user can change application settings as well as database specific settings.

Edit->Item Types

Opens the “Preferences Dialog” to the tab that gives the user options to create and modify the item types supported by the database.

Edit->Taxonomies

Opens the “Preferences Dialog” to the tab that gives the user options to create and modify the taxonomies supported by the database.

Edit->Preferences (Ctrl+P)

Opens the “Preferences Dialog”. Here the user can change application settings as well as database specific settings.
