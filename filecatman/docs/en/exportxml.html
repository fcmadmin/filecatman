Export XML

With this wizard, a Filecatman XML file (*.xml) can be created from the project data. The types of data that can be stored in the file, are items, categories, relations, item types and taxonomies. At the end of the export process a finish page will be shown. The XML could then be imported back into a project, or it could be packaged with the data folder and the quick installers. The quick installer does the same function as the Create Symbolic Links Wizard, but doesn't require Filecatman. The quick installers are included in the source code.
